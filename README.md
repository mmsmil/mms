## Що це

MMS — закрите програмне забезпечення для командирів підрозділів, діловодів і штабних працівників. Замінює Excel-таблиці, паперові журнали і ручну роботу зі списками особового складу.

Повністю офлайн. Всі дані зберігаються локально та зашифровані. Ніяких хмарних сервісів, ніяких акаунтів.

---

## Можливості

### Особовий склад
Облік усього підрозділу: ПІБ, звання, посада, дата народження, телефон, фото. Кастомні поля і таблиці під специфіку підрозділу. Пошук, фільтрація, групи. Експорт у Excel та Word-документи.

### Графік нарядів
Місячна сітка призначень. Групи чергування, типи нарядів. Відпустки і свята. Drag-and-drop редагування, undo/redo. Звіт по службі у форматі DOCX.

### Документи
Шаблони Word із плейсхолдерами — дані підставляються автоматично з бази. Генерація наказів, списків, довідок. Підтримка таблиць і умовних блоків у шаблонах.

### Дашборд
Зведена картина підрозділу: стан боєздатності, навантаження, відпустки, дні народження. Персональна картка кожного — хронологія служби та активність за рік.

### Резервне копіювання
Ручне та автоматичне. Вибіркове відновлення окремих таблиць. Сумісність зі старішими версіями бекапів.

## Встановлення

1. Завантаж інсталятор з [Releases](https://github.com/mmsmil/mms/releases/latest)
2. Запусти `MMS_1.1.1_x64-setup.exe`
3. Пройди початкове налаштування (EULA → пароль адміна → готово)

**Системні вимоги:** Windows 10/11 x64 · ~50 MB на диску

> **Якщо Windows показує "Захисник Windows заблокував запуск":**
> Натисни **Детальніше** → **Все одно запустити**
